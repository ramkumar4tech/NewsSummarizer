# NewsSummarizer
AI Agent to summarize news article

Pre-Requisites :
  Packages to be installed :
    pip install langgraph langchain_ollama crawl4ai pandas openpyxl
    playwright install

  GMail App Password :
    [https://myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords)

  Create environment file :
    EMAIL_ADDRESS=
    EMAIL_PASSWORD=
    RECIPIENT_EMAIL=
